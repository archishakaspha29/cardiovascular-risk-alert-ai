# Dataset Sources

This project uses public cardiovascular health datasets for educational and research purposes.

## Public Dataset Sources

- Kaggle cardiovascular and heart disease datasets
- Heart failure prediction dataset
- Cardiovascular disease training dataset

## Data Use Notes

The datasets used in this project are public datasets and do not contain private patient information from this project.

The model results are based on available public data and should not be treated as clinical validation.

## Data Limitations

Public datasets may not fully represent all populations. Some groups may be underrepresented due to gaps in healthcare data collection. Future work should include testing with larger, more diverse, and medically reviewed datasets.
