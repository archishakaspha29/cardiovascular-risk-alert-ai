# Project Summary

## Project Title

AI-Assisted Cardiovascular Risk-Alert Tool for Underserved Communities

## Purpose

This project explores how machine learning can support early cardiovascular risk awareness for underserved communities. The tool is designed to help users better understand possible cardiovascular risk factors and recognize when they may need to seek medical care.

This is a risk-alert and health-awareness prototype. It is not a medical diagnosis tool and does not replace a physician.

## Problem

Many low-income families and underserved communities may face delays in accessing healthcare. For cardiovascular conditions, delayed care can increase health risks. This project aims to support earlier care-seeking by using public health data and machine learning to identify possible risk patterns.

## Solution

The project uses a Random Forest machine learning model to analyze cardiovascular health indicators and classify possible risk. The model also uses feature importance to help explain which health factors contributed most to the risk output.

## Key Features

- Cardiovascular risk-alert prediction
- Public dataset-based model training
- Accuracy, precision, recall, and F1-score evaluation
- Feature importance for explainability
- Bias and fairness considerations
- Ethical AI guardrails
- Focus on underserved healthcare access

## Ethical AI Considerations

This project considers bias, underrepresentation, fairness, transparency, privacy, and responsible use of demographic information. Race and demographic variables are treated carefully because they may reflect social, environmental, and healthcare-access disparities rather than direct biological causes.

## Current Status

This project is currently a prototype. Future work includes improving the user interface, testing with larger and more diverse datasets, adding stronger fairness metrics, getting medical expert feedback, and building a safer web or mobile app version.

## Long-Term Goal

The long-term goal is to develop this into a responsible cardiovascular risk-awareness app that supports underserved communities while following medical safety, privacy, and ethical AI guidelines.
