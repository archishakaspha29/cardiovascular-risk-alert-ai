This folder contains model output visualizations generated by `src/random_forest_model.py`.

Files:
- heart_classification.png / .svg / .pdf — Classification metrics (precision/recall/f1) for Dataset 1.
- cardio_train_classification.png / .svg / .pdf — Classification metrics (precision/recall/f1) for Dataset 2.
- heart_feature_importance.png / .svg / .pdf — Feature importance bar chart for Dataset 1.

How to regenerate:
```bash
MPLBACKEND=Agg python3 src/random_forest_model.py
```

Notes:
- Files are saved in PNG, SVG and PDF formats for high-quality publication and quick previews.
- If you prefer different names or formats, edit `src/random_forest_model.py` and rerun the script.
